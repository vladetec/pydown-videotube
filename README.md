# Pydown-VideoTube
E um pequeno exemplo com código simplista para baixar vídeo do youtube usando Python.

## Rodar o Projeto

```shell
git clone https://github.com/vladetec/pydown-videotube.git
cd pydown-videotube
python3 -m venv venv
source venv/bin/activate
pip install -U pip
pip install -r requirements.txt
python app.py 
```
